Example

Demonstration of the different border widths:

p.one {
        border-style:solid;
        border-width: 5px;
    }
    

    p.two {
        border-style: solid;
        border-width: medium;
    }

    p.three {
        border-style: dotted;
        border-width: 2px;
    }


    p.four {
        border-style: dotted;
        border-width: thick;
    }
